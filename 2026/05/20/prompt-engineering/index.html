原则1：指令清晰且具体

  • 使用分隔符区分文字的不同部分

    • ‘’’
    • ```
    • -–
    • <
    • >

  • 明确模型的任务：your task is…

  • 要求模型按照指定格式输出

    • JSON格式，包含时间、地点、任务这几个key
    • HTML

  • 提前假设条件

    • 如果不满足，则…

  • Few-shot提示

    • 给几个示例
